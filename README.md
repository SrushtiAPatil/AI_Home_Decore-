# AI Home Decor Application

## Overview
AI Home Decor is an Android-based application developed using Java and Kotlin that helps users visualize and plan interior designs before making any changes to their living spaces. The application enables users to upload room images, virtually place furniture and decorative objects, and explore different design possibilities in a realistic environment.

In addition, the system provides AI-based design suggestions and recommends suitable décor templates based on room layout and user preferences. This helps users make informed decisions, save time, reduce costs, and improve the overall home decoration experience.

## Features
- Upload room images
- Virtual furniture placement
- AI-based room design suggestions
- Interior design templates
- Realistic room visualization

## Technologies Used
- Java
- Kotlin
- Android Studio
- XML
- Image Processing
